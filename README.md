# Learning Journal

A personal learning journal application built with Vite, JavaScript, CSS, and HTML.

## About

Learning Journal is an interactive web application designed to document and track your learning journey. It allows you to create posts about your learning experiences, view recent posts, and organize your educational progress in one convenient location.

## Tech Stack

- **Frontend:** JavaScript (61.8%), HTML (22.6%), CSS (15.6%)
- **Build Tool:** Vite
- **Language:** JavaScript, CSS, HTML

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

Install the dependencies:

```bash
npm install
# or
yarn install
```

### Development

Run the development server:

```bash
npm start
# or
yarn start
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).

## Features

- ✔ Create and manage learning posts
- ✔ Custom images and personalized content
- ✔ Document your learning experiences
- ✔ Multiple pages with Blog View
- ✔ View recent posts with "View More" functionality
- ✔ Responsive navigation with hamburger menu for mobile devices
- ✔ Clean and intuitive user interface

## How to Use

1. Start the application
2. Create new journal entries about your learning
3. Add custom images and descriptions
4. Organize your posts by date
5. View recent posts on the main page
6. Navigate through your learning history with the responsive menu

## Build for Production

```bash
npm run build
# or
yarn build
```
